Кольца, поля и модулярная арифметика

Одной операции мало: шифрование перемножает, подписи складывают. Эта глава достраивает вторую операцию и доказывает то, что глава 1 использовала без доказательства: критерий обратимости, функцию Эйлера и теоремы, обеспечивающие корректность RSA.

2.0Две операции

В группе одна операция. Но уже RSA-шифрование \(m \mapsto m^e \bmod n\) перемножает остатки, а его анализ требует и складывать их: показатели степеней приводятся по модулю функции Эйлера \(\varphi(n)\), доказательства манипулируют суммами. Структура, в которой сложение и умножение согласованы, называется кольцом; кольцо, в котором каждый ненулевой элемент обратим по умножению, — полем.

Определение

Коммутативное кольцо с единицей — множество \(R\) с двумя операциями \((+, \cdot)\), где:

  1. \((R, +)\) — абелева группа (нейтральный элемент — \(0\));
  2. умножение ассоциативно, коммутативно, имеет нейтральный элемент \(1\);
  3. дистрибутивность: \(a \cdot (b + c) = a \cdot b + a \cdot c\).

Поле — коммутативное кольцо с единицей, в котором \(0 \ne 1\) и каждый ненулевой элемент обратим по умножению.

\(\mathbb{Z}\) — кольцо, но не поле. \(\mathbb{Q}\) и \(\mathbb{R}\) — поля. \(\mathbb{Z}_n\) со сложением и умножением по модулю — кольцо для любого \(n \ge 2\); является ли оно полем — зависит от \(n\), и это центральный вопрос главы. В \(\mathbb{Z}_6\) равенство \(2 \cdot 3 = 0\) показывает новое явление: произведение ненулевых элементов равно нулю. Такие элементы называются делителями нуля, и они несовместимы с обратимостью: .

2.1Алгоритм Евклида и лемма Безу

Критерий из главы 1 — «обратимы те \(a\), у которых \(\gcd(a, n) = 1\)» — остался без доказательства. Инструмент для него старше самой криптографии. НОД двух чисел вычисляется без разложения на множители: — алгоритм Евклида. Если во время спуска запоминать частные, тем же проходом получается больше, чем НОД: — лемма Безу.

Лемма Безу даёт недостающее доказательство: . Отсюда же — ответ на вопрос о полях: . Над конечными полями \(\mathbb{Z}_p\) строятся эллиптические кривые (глава 5).

в криптографии

Генерация RSA-ключа заканчивается вычислением секретной экспоненты \(d = e^{-1} \bmod \varphi(n)\) — это один запуск расширенного алгоритма Евклида. Он же вычисляет обратные элементы в формулах эллиптических кривых (глава 5) и при декодировании в постквантовых схемах (глава 9).

2.2Китайская теорема об остатках

Следующий инструмент разбирает кольцо \(\mathbb{Z}_{mn}\) на независимые компоненты. Число \(x \in \mathbb{Z}_{mn}\) даёт два остатка: \(x \bmod m\) и \(x \bmod n\). Пара остатков определяет число однозначно: — китайская теорема об остатках (CRT). Работать с парой остатков можно покомпонентно, а в конце собрать результат обратно в \(\mathbb{Z}_{mn}\).

в криптографии

RSA-расшифрование \(c^d \bmod n\) при \(n = pq\) на практике выполняется через CRT: два возведения в степень по модулям \(p\) и \(q\) вместо одного по модулю \(n\) — примерно в четыре раза быстрее, поскольку стоимость умножения растёт квадратично с длиной чисел. Так делает каждая библиотека TLS.

2.3Функция Эйлера

Функция Эйлера \(\varphi(n)\) — количество обратимых элементов в \(\mathbb{Z}_n\), то есть порядок группы \(\mathbb{Z}_n^{*}\). По критерию обратимости это число элементов \(\{1, \dots, n\}\), взаимно простых с \(n\). Для простого \(p\) все ненулевые остатки обратимы: \(\varphi(p) = p - 1\). Для степени простого , а для взаимно простых сомножителей — мультипликативность, прямое следствие CRT. Комбинация этих фактов даёт формулу для произвольного \(n\) по его разложению: \[ \varphi(n) \;=\; \prod_{i} \left(p_i^{k_i} - p_i^{k_i - 1}\right) \;=\; n \prod_{p \mid n} \Bigl(1 - \frac{1}{p}\Bigr). \]

Формула требует знать разложение \(n\). Для RSA-модуля \(n = pq\) это принципиально: \(\varphi(n) = (p-1)(q-1)\) мгновенно вычисляется тем, кто знает \(p\) и \(q\), и неизвестно тому, кто видит только \(n\). Знание \(\varphi(n)\) равносильно факторизации: из \(n\) и \(\varphi(n)\) числа \(p, q\) восстанавливаются как корни квадратного уравнения \(t^2 - (n - \varphi(n) + 1)t + n = 0\).

2.4Теоремы Эйлера и Ферма

Все компоненты собраны: \(\mathbb{Z}_n^{*}\) — группа (обратимость доказана в 2.1), её порядок — \(\varphi(n)\) (2.3), а по следствию теоремы Лагранжа (глава 1) элемент группы в степени порядка группы равен нейтральному. Отсюда — теорема Эйлера; частный случай простого \(n = p\) — малая теорема Ферма: \(a^{p-1} \equiv 1 \pmod p\) для \(a \not\equiv 0\).

Осталась вычислительная сторона: почему возведение в степень — «дешёвая» операция. Наивное умножение \(m \cdot m \cdots m\) для криптографических показателей (\(e \sim 2^{2048}\)) невыполнимо, но разложением показателя по двоичной записи (square-and-multiply). Итог — асимметрия: возведение в степень полиномиально по длине входа, а для обращения (дискретный логарифм, факторизация) полиномиальный алгоритм неизвестен.

в криптографии

Схема RSA целиком: ключи \(n = pq\), открытая экспонента \(e\) с \(\gcd(e, \varphi(n)) = 1\), секретная \(d = e^{-1} \bmod \varphi(n)\); шифрование \(c = m^e \bmod n\), расшифрование \(c^d \bmod n\). Корректность — это равенство \(m^{ed} = m\) в \(\mathbb{Z}_n\): для обратимых \(m\) оно следует из теоремы Эйлера (\(ed = 1 + k\varphi(n)\)), но — доказательство склеивает случаи по CRT. Полный разбор схемы, включая обязательный padding, — в главе 4.

2.5Что дальше

Математика асимметричных схем построена, но прежде чем собирать из неё протоколы, нужна симметричная половина криптографии: блочные шифры и хэш-функции. Без хэшей не определить подпись (подписывается не сообщение, а его хэш), без симметричных шифров не объяснить, зачем протоколам обмен ключами и как квантовые атаки действуют на разные примитивы по-разному. Лемма

Пусть \(a \in \mathbb{Z}_n\), \(a \ne 0\). Если \(\gcd(a, n) = 1\), то \(a\) обратим; иначе \(a\) — делитель нуля. Эти случаи несовместимы.

Доказательство. Случай \(\gcd(a, n) = 1\) — критерий обратимости (сноска в 2.1). Пусть \(d = \gcd(a, n) > 1\). Возьмём \(b = n/d\): тогда \(b \ne 0\) в \(\mathbb{Z}_n\) (так как \(0 < b < n\)) и \[ a \cdot b = a \cdot \frac{n}{d} = \frac{a}{d} \cdot n \equiv 0 \pmod n, \] где \(a/d\) — целое. Значит, \(a\) — делитель нуля. Совместить случаи нельзя: если \(a\) обратим и \(ab \equiv 0\), умножение на \(a^{-1}\) даёт \(b \equiv 0\) — противоречие с \(b \ne 0\).
доказано
Лемма

Для \(b \ne 0\): \(\gcd(a, b) = \gcd(b,\; a \bmod b)\). Повторение шага завершается, и число делений есть \(O(\log \min(a, b))\).

Доказательство. Пусть \(r = a - qb\), где \(q = \lfloor a/b \rfloor\). Любой общий делитель \(a\) и \(b\) делит \(r = a - qb\); любой общий делитель \(b\) и \(r\) делит \(a = qb + r\). Множества общих делителей пар \((a, b)\) и \((b, r)\) совпадают — значит, совпадают и наибольшие элементы.

Завершение: остатки убывают строго (\(0 \le r < b\)), достигая нуля. Оценка: если \(a \ge b\), то \(a \bmod b < a/2\) (при \(b \le a/2\) это следует из \(r < b\); при \(b > a/2\) частное равно \(1\) и остаток \(a - b < a/2\)). Каждые два шага меньший аргумент уменьшается вдвое, итого \(O(\log \min(a,b))\) делений.
доказано
Лемма · Безу

Для любых целых \(a, b \ge 0\) существуют целые \(x, y\) такие, что \(a x + b y = \gcd(a, b)\).

Доказательство. Индукция по числу шагов алгоритма Евклида.

База (\(b = 0\)): \(\gcd(a, 0) = a = a \cdot 1 + 0 \cdot 0\), годятся \(x = 1\), \(y = 0\).

Шаг. Пусть для пары \((b,\; a \bmod b)\) коэффициенты найдены: \[ b x' + (a \bmod b)\, y' = g, \qquad g = \gcd(a, b). \] Подставим \(a \bmod b = a - \lfloor a/b \rfloor\, b\): \[ g = a\, y' + b\,\bigl(x' - \lfloor a/b \rfloor\, y'\bigr), \] то есть \(x = y'\), \(y = x' - \lfloor a/b \rfloor\, y'\). Обратный ход в виджете — в точности эта подстановка, выполненная от базы вверх.
доказано
Теорема

\(a\) обратим в \(\mathbb{Z}_n\) \(\iff\) \(\gcd(a, n) = 1\). Обратный элемент вычисляется за \(O(\log n)\) арифметических операций.

Доказательство. (⇐) По лемме Безу \(a x + n y = 1\) для некоторых целых \(x, y\). По модулю \(n\): \(a x \equiv 1 \pmod n\), значит \(a^{-1} = x \bmod n\).

(⇒) Если \(d = \gcd(a, n) > 1\) и \(a x \equiv 1 \pmod n\), то \(a x - 1 = q n\), откуда \(1 = a x - q n\) делится на \(d\) — противоречие.

Вычисление: расширенный алгоритм Евклида — \(O(\log n)\) делений (сноска «алгоритм Евклида»).
доказано

Это доказательство закрывает сноску «обратимость в \(\mathbb{Z}_n\)» главы 1, где лемма Безу использовалась как чёрный ящик.

Теорема

\(\mathbb{Z}_n\) — поле \(\iff\) \(n\) простое.

Доказательство. (⇐) Пусть \(n = p\) простое. Для любого \(a \in \{1, \dots, p-1\}\): \(\gcd(a, p) \in \{1, p\}\), а \(p \nmid a\), значит \(\gcd(a, p) = 1\) и \(a\) обратим по критерию обратимости. Все ненулевые обратимы — поле.

(⇒) Пусть \(n\) составное: \(n = uv\), \(1 < u, v < n\). Тогда \(\gcd(u, n) = u > 1\), элемент \(u\) необратим (критерий), и \(\mathbb{Z}_n\) — не поле.
доказано
Теорема · CRT

Пусть \(\gcd(m, n) = 1\). Отображение \(\psi(x) = (x \bmod m,\; x \bmod n)\) — биекция \(\mathbb{Z}_{mn} \to \mathbb{Z}_m \times \mathbb{Z}_n\), причём \(\psi(x + y) = \psi(x) + \psi(y)\) и \(\psi(x y) = \psi(x)\,\psi(y)\) (покомпонентно).

Доказательство. Согласованность с операциями — свойство взятия остатка: \((x + y) \bmod m = (x \bmod m + y \bmod m) \bmod m\), аналогично для умножения и для модуля \(n\).

Инъективность. Пусть \(\psi(x) = \psi(y)\), то есть \(m \mid x - y\) и \(n \mid x - y\). Запишем \(x - y = k m\). По лемме Безу \(1 = m u + n v\), умножим на \(k\): \[ k = k m u + k n v = (x - y) u + k n v. \] Оба слагаемых делятся на \(n\) (первое — так как \(n \mid x - y\)), значит \(n \mid k\), и \(x - y = km\) делится на \(mn\). В \(\mathbb{Z}_{mn}\) это означает \(x = y\).

Биекция. Инъекция между конечными множествами равной мощности \(|\mathbb{Z}_{mn}| = mn = |\mathbb{Z}_m \times \mathbb{Z}_n|\) — биекция.

Явное обращение. Для пары \((r_1, r_2)\): \[ x = \bigl(r_1 \cdot n \cdot (n^{-1} \bmod m) + r_2 \cdot m \cdot (m^{-1} \bmod n)\bigr) \bmod mn. \] Первое слагаемое \(\equiv r_1 \pmod m\) и \(\equiv 0 \pmod n\); второе — наоборот. Обратные существуют по критерию обратимости (\(\gcd(m, n) = 1\)).
доказано
Лемма

\(\varphi(p^k) = p^k - p^{k-1}\) для простого \(p\).

Доказательство. \(\gcd(a, p^k) > 1\) \(\iff\) \(p \mid a\). Среди \(\{1, \dots, p^k\}\) кратных \(p\) ровно \(p^{k-1}\) штук (\(p, 2p, \dots, p^{k-1} \cdot p\)). Остальные \(p^k - p^{k-1}\) взаимно просты с \(p^k\).
доказано
Теорема

Если \(\gcd(m, n) = 1\), то \(\varphi(mn) = \varphi(m)\,\varphi(n)\).

Доказательство. Биекция CRT переводит обратимые элементы в пары обратимых: \[ \gcd(x, mn) = 1 \iff \gcd(x, m) = 1 \text{ и } \gcd(x, n) = 1 \] (простой делитель, общий у \(x\) и \(mn\), делит \(m\) или \(n\); обратно, общий делитель \(x\) и \(m\) делит и \(mn\); остатки сохраняют НОД: \(\gcd(x, m) = \gcd(x \bmod m, m)\)). Значит, \(\psi\) ограничивается до биекции \(\mathbb{Z}_{mn}^{*} \to \mathbb{Z}_m^{*} \times \mathbb{Z}_n^{*}\), и мощности равны: \(\varphi(mn) = \varphi(m)\,\varphi(n)\).
доказано
Теорема · Эйлер

Если \(\gcd(a, n) = 1\), то \(a^{\varphi(n)} \equiv 1 \pmod n\).

Доказательство 1 (через Лагранжа). \(\mathbb{Z}_n^{*}\) — группа порядка \(\varphi(n)\); по следствию теоремы Лагранжа (глава 1) \(a^{|G|} = e\) для любого элемента, то есть \(a^{\varphi(n)} \equiv 1\).
доказано
Доказательство 2 (прямое, без Лагранжа). Пусть \(u_1, \dots, u_{\varphi(n)}\) — все элементы \(\mathbb{Z}_n^{*}\). Умножение на \(a\) — перестановка \(\mathbb{Z}_n^{*}\) (правило сокращения, глава 1), поэтому наборы \(\{u_i\}\) и \(\{a u_i\}\) совпадают как множества. Перемножим каждый набор целиком: \[ \prod_i u_i \;\equiv\; \prod_i (a u_i) \;=\; a^{\varphi(n)} \prod_i u_i \pmod n. \] Произведение \(\prod u_i\) обратимо (как произведение обратимых), сокращаем: \(1 \equiv a^{\varphi(n)}\).
доказано
Следствие · Ферма

Для простого \(p\) и \(a \not\equiv 0 \pmod p\): \(a^{p-1} \equiv 1 \pmod p\).

Доказательство. \(\varphi(p) = p - 1\), и \(\gcd(a, p) = 1\) для \(a \not\equiv 0\).
Лемма

\(a^e \bmod n\) вычисляется не более чем за \(2\lfloor\log_2 e\rfloor + 1\) умножений по модулю \(n\).

Доказательство. Запишем \(e = \sum_i e_i 2^i\) в двоичной системе. Тогда \[ a^e = \prod_{i : e_i = 1} a^{2^i}. \] Последовательность \(a^{2^0}, a^{2^1}, a^{2^2}, \dots\) вычисляется последовательным возведением в квадрат: \(a^{2^{i+1}} = (a^{2^i})^2\) — по одному умножению на бит. Накопление произведения по единичным битам — ещё не более одного умножения на бит. Все операции — по модулю \(n\), промежуточные значения не превышают \(n^2\).
доказано

Для \(e \sim 2^{2048}\) это ≈ 4000 умножений вместо \(2^{2048}\). Замечание для практики: наивная реализация с ветвлением по биту раскрывает секретный показатель через время выполнения (timing attack) — реальные библиотеки используют constant-time варианты. Подробнее в главе 4.

Теорема

Пусть \(n = pq\), \(p \ne q\) простые, \(ed \equiv 1 \pmod{\varphi(n)}\). Тогда \(m^{ed} \equiv m \pmod n\) для всех \(m \in \mathbb{Z}_n\).

Доказательство. Запишем \(ed = 1 + k\varphi(n) = 1 + k(p-1)(q-1)\). По CRT достаточно проверить сравнение по модулям \(p\) и \(q\) по отдельности.

Модуль \(p\). Если \(p \mid m\), обе части сравнимы с \(0\). Иначе \(\gcd(m, p) = 1\), и по малой теореме Ферма \[ m^{ed} = m \cdot \bigl(m^{p-1}\bigr)^{k(q-1)} \equiv m \cdot 1^{k(q-1)} = m \pmod p. \] Модуль \(q\) — симметрично.

Итак, \(p \mid m^{ed} - m\) и \(q \mid m^{ed} - m\); по инъективности CRT (или напрямую: \(pq \mid m^{ed} - m\) для взаимно простых \(p, q\)) получаем \(m^{ed} \equiv m \pmod n\).
